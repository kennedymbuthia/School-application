## 1. Getting Started

### 1.1 Login
1. Open the school management system URL
2. Enter your email and password
3. Click "Login"
4. If you forgot password, click "Forgot Password"

### 1.2 First Time Login

- Update your profile information
- Verify your contact details

### 1.3 Navigation
- **Dashboard**: Overview of key information
- **Sidebar**: Access all features
- **Profile**: Update personal details

## 2. For Parents

### 2.1 Viewing Your Child's Information
After login, parents can view:

#### View Grades
1. Go to **My Children** → **Select Child**
2. Click **Grades** tab
3. View:
   - Current grades by subject
   - Grade history
   - Teacher comments

#### View Attendance
1. Go to **My Children** → **Select Child**
2. Click **Attendance** tab
3. View:
   - Daily attendance record
   - Attendance percentage
   - Absence reasons

#### View Timetable
1. Go to **Timetable**
2. View:
   - Weekly class schedule
   - Subject details
   - Room assignments

### 2.2 Making Payments

#### Online Payment (M-Pesa)
1. Go to **Payments** → **Make Payment**
2. Select fee type
3. Enter amount
4. Choose payment method:
   - **M-Pesa**: Follow prompts on your phone
   - **Bank Transfer**: View bank details
   - **Cash**: Visit school office
5. Confirm payment
6. Download receipt

#### Payment History
1. Go to **Payments** → **History**
2. View all past payments
3. Download receipts as PDF

### 2.3 Communication

#### Viewing Announcements
1. Go to **Announcements**
2. View:
   - School notices
   - Event notifications
   - Important alerts

#### Contacting Teachers
1. Go to **Messages**
2. Click "New Message"
3. Select recipient (teacher)
4. Type your message
5. Send

### 2.4 Profile Management
1. Go to **Profile**
2. Update:
   - Phone number
   - Email address
   - Physical address
3. Click "Save Changes"

## 3. For Teachers

### 3.1 Dashboard Overview
Teacher dashboard shows:
- My classes
- Today's schedule
- Pending grade submissions
- Announcements

### 3.2 Taking Attendance

#### Daily Attendance
1. Go to **Attendance** → **Mark Attendance**
2. Select:
   - Class
   - Date (defaults to today)
3. Mark each student:
   - Present (green)
   - Absent (red)
   - Late (yellow)
   - Excused (blue)
4. Add remarks if needed
5. Click "Save"
6. Click "Lock" to prevent changes

#### Viewing Attendance History
1. Go to **Attendance** → **History**
2. Filter by:
   - Class
   - Date range
3. View summary statistics

### 3.3 Managing Grades

#### Entering Grades
1. Go to **Grades** → **Enter Grades**
2. Select:
   - Class
   - Subject
   - Term/Exam type
3. Enter scores for each student
4. Click "Save as Draft" (to edit later)
5. Click "Submit" (for approval)

#### Grade Approval Status
- **Draft**: Not yet submitted
- **Submitted**: Awaiting admin approval
- **Approved**: Published to parent portal
- **Locked**: Finalized, no changes allowed

### 3.4 Timetable

#### Viewing Schedule
1. Go to **Timetable**
2. View:
   - Weekly teaching schedule
   - Assigned classes/subjects
   - Room allocations

### 3.5 Class Management

#### Viewing Class List
1. Go to **My Classes**
2. Select class
3. View:
   - Student list
   - Parent contacts
   - Class details

### 3.6 Messages & Communication

#### Sending Messages to Parents
1. Go to **Messages**
2. Click "New Message"
3. Select recipient(s):
   - Individual parent
   - Entire class
4. Type message
5. Send

#### Receiving Messages
1. Go to **Inbox**
2. View messages from:
   - Parents
   - Admin
   - Other teachers
3. Reply directly

## 5. Help & Support

### 5.1 Common Issues

#### Cannot Login
- Check email/password
- Clear browser cache
- Use "Forgot Password"

#### Payment Failed
- Verify sufficient balance
- Check M-Pesa/USSD limits
- Try alternative method

#### Grade Not Saving
- Check required fields
- Verify numeric values
- Contact admin if persists

### 5.2 Password Reset
1. Click "Forgot Password"
2. Enter email
3. Check inbox for reset link
4. Create new password
5. Login with new password

### 5.3 Contact Support
- **Email**: support@school.edu

- **Hours**: Mon-Fri, 8am-5pm
